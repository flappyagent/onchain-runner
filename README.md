# Onchain Runner

An endless runner Web3 game integrated with the Base blockchain. Players compete for highest on-chain scores.

## Features
- Score tracking contract
- Wallet-based authentication
- Leaderboard integration

## Technologies
Solidity  
Base  
ethers.js
